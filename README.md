# SmartFuel Route Optimizer

A Data Structures & Algorithms project that computes fuel-efficient routes and ranks nearby fuel stations using Graphs, Dijkstra’s Algorithm, BFS, Max-Heaps, and Spatial Grid Mapping.

## Features
- Optimal route calculation
- Fuel station recommendation
- Spatial request filtering
- Detour scoring
- Heap-based ranking system

## Data Structures Used
- Graphs
- Arrays
- Max-Heaps
- Queues
- HashMaps
- 2D Grids

## Algorithms Used
- Dijkstra’s Algorithm
- BFS
- Heap Operations
- Spatial Mapping

## Tech Stack
- Java
- DSA Concepts
- Terminal-based UI
